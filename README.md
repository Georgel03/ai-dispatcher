# AI Dispatcher - Smart Fleet Management System

A full-stack logistics platform designed to optimize freight dispatching using AI and real-time tracking. Built as a Bachelor's Thesis project.

## 🏗 Architecture
* **Frontend:** Next.js 14 (React), TailwindCSS, Clerk Auth
* **Backend:** FastAPI (Python), SQLAlchemy, Pydantic
* **Database:** PostgreSQL (Dockerized)
* **Infrastructure:** Docker & Docker Compose
